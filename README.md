# VoxScript Privacy Policy

This repository contains the public-facing privacy policy for VoxScript iOS app.

**View Privacy Policy:** [privacy-policy.html](privacy-policy.html)

## About VoxScript

VoxScript is an offline AI transcription app for iOS. All transcription happens on-device with complete privacy.

- **No data collection**
- **No cloud uploads**
- **No tracking**
- **One-time purchase, no subscriptions**
